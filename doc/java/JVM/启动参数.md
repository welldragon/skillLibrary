# 内存

|内存类型|参数|含义|
|---|---|---|
|Heap|-Xms|初始和最小的heap大小
||-Xmx|最大的heap大小
|Metaspace|-XX:MaxMetaspaceSize|最大的Metaspace大小，不设置默认为不限制
|Stack|-Xss|线程stack大小
|CodeCache|-XX:ReservedCodeCacheSize|最大的code cache大小


JDK8之后已经没有PermGen，换成了Metaspace
